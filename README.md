## Getting Started
### Instaling
**Clone the repository**
```
$git clone https://github.com/RenanTRS/Cronometro.git
$cd Cronometro
```
**Instaling dependencies**
```
$yarn
```
or
```
$npm install
```
## Running
With all dependencies installed and the environment properly configured, you cannow run the project:
```
$yarn start
```
or
```
$npm start
```
